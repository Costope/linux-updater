# Linux Updater

A simple bash tool to update your Debian(-Distrubution)

## System tested:
- Ubuntu 22.04 LTS

## Installation

Clone that repository:
```bash
git clone https://github.com/costope/linux-updater.git
```

Change directory:
```bash
cd linux-updater/Updater
```

Run the script:
```bash
sh ./updater.sh
```
